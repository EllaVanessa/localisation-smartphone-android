# Application Android — Localisation Smartphone

## Description

Application Android développée avec Java permettant de :

- récupérer la position GPS ;
- afficher les coordonnées ;
- envoyer les données vers un serveur PHP ;
- enregistrer les positions dans MySQL.

---

# Technologies utilisées

- Java
- Android Studio
- Volley
- PHP
- MySQL
- XAMPP

---

# Fonctionnement

Android → PHP → MySQL
